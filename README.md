# AI Deforestation Monitoring

This repository contains the code for the AI Deforestation Monitoring project.

Milestone 4 · Phase 4 scaffold (backend-first).

Quick start (backend-only, development):

1. Copy .env.example to .env and edit values as needed.

2. Start the stack:

```bash
docker compose up --build
```

3. The API will be available at http://localhost:8000
   - Health: GET /api/health
   - Predict: POST /api/predict (JSON sample documented below)
